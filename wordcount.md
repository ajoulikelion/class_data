## MTV 패턴

Django는 MTV(Model - Template - View)를 기반으로 한 프레임워크입니다. MTV 패턴이란 데이터, 사용자 인터페이스, 데이터를 처리하는 로직을 구분해서 한 요소가 다른 요소들에 영향을 주지 않도록 설계하는 방식입니다. 이러한 방식으로 개발을 진행하면 UI 디자이너와 개발자는 독립적인 영역에서 개발이 가능하게 됩니다.
